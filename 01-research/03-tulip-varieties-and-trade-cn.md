# 郁金香品种与贸易：历史准确性评估

> 本报告编制于2026年5月，服务于 Tulip Mania 1637 游戏项目。报告旨在评估游戏中所用郁金香品种的历史准确性。

---

## 1. 游戏中的郁金香品种与历史对照

### 品种准确性评估

| 游戏品种 | 历史准确性 | 备注 |
|---|---|---|
| **Semper Augustus** | ✅ 准确 | 最负盛名、价格最高的郁金香品种。白色底色上缀以红色火焰纹。确有其品种，且为当时的顶级名品。 |
| **Viceroy** | ✅ 准确 | 仅次于 Semper Augustus 的名种。在 Pieter Cos[^1] 1637年花木目录中有详实记载。花瓣呈深紫色/紫罗兰色火焰纹。 |
| **Gouda** | ⚠️ 存疑 | "Gouda" 并非1630年代有文献记载的郁金香品种名称。Gouda 市确实是郁金香交易中心之一，但主要花木目录中并无名为"Gouda"的具体品种。**建议更改为有文献记载的品种。** |
| **Black Tulip**（黑郁金香） | ❌ 时代错误 | 1637年并不存在真正的黑色郁金香——第一株真正的黑郁金香培育于**1986年**，晚了350年之久。当时的深色郁金香实为深紫色或栗色（Couleren[^2] 类）。所谓"寻找黑郁金香"的传说源自 Dumas[^3] 1850年的小说。**应更名为当时实际存在的深色品种，或明确说明其为深紫色。** |

### 推荐替代品种

#### 替代"Gouda"
以下为有历史文献记载的可替代品种：
- **Admirael van der Eijck** — 在 Pieter Cos 目录中有详实记载；1637年2月5日以1,045 guilders[^4] 成交
- **Admirael Liefkens** — 高档品种
- **Generalissimo** — 高档品种（但文献记载较少）
- **Paragon d'Utrecht** — 以 Utrecht 命名
- **Groot Geel en Rooi van Leyde** — 意为"Leiden 的大黄红"

#### 替代"Black Tulip"
以下为有历史文献记载的深色品种：
- **Prince van Astrijk**（奥地利亲王）— 深色品种
- **Zwarte Tulip** — 虽然深色郁金香确实存在，但"黑色"这一特定称谓属于时代错误
- 或直接将其描述为一种**"深紫色 Couleren"**品种

## 2. Semper Augustus — 完整历史档案

### 外观描述
- **花色**：明艳的白色底色，花瓣上分布着鲜艳的红色"火焰"或"羽毛"条纹
- **类型**：Rosen[^5] 类（白色底色配红色/粉色火焰纹）
- **花纹成因**：由 Tulip Breaking Virus[^6]（郁金香碎色病毒）引起

### 价格沿革

| 年份 | 价格 | 备注 |
|---|---|---|
| ~1623 | ~1,000 guilders | 早期有记录的价格 |
| ~1625 | ~1,200 guilders | 有人出价但被持有者拒绝出售 |
| ~1633 | ~5,500 guilders | 文献记载的估值 |
| 1637年高峰 | 可能达到 ~6,000+ guilders | 狂潮顶峰时期 |
| 崩盘后 | ~10 guilders 或更低 | 跌幅超过98% |

### 著名拥有者：Adriaen Pauw[^7]
- **Holland 大议长**（Grand Pensionary of Holland[^8]），当时最重要的政治人物之一
- 拥有 Haarlem 附近的 **Heemstede** 庄园
- 藏有著名的 Semper Augustus 系列
- 据传**拒绝以任何价格出售**——将其视为珍宝，而非商品

### 何以如此昂贵
1. **Tulip Breaking Virus** 造就了花纹，但也削弱了植株
2. 每一株碎色郁金香都是**独一无二**的——花纹无法预测
3. 病毒**减少了子球的繁殖**——供给不断萎缩
4. 无法可靠地人工复制——从种子培育需5至12年，且极少出现"碎色"
5. 极端的**稀缺性**遭遇极端的需求

## 3. Viceroy — 完整历史档案

### 外观描述
- **类型**：Violetten[^9] 或 Bizarden[^10] 类
- 浅色底色上分布深紫色/紫罗兰色火焰纹
- 花朵硕大

### 著名的易物交易
Pieter Cos 1637年目录记载了一颗 Viceroy 球茎换得的物资：

| 物品 | 数量 |
|---|---|
| 小麦（Last[^11]） | 4 |
| 黑麦（Last） | 8 |
| 肥牛 | 4 |
| 肥猪 | 8 |
| 肥羊 | 12 |
| 葡萄酒（Hogshead[^12]） | 2 |
| 黄油（吨） | 4 |
| 奶酪（千磅） | 1 |
| 床铺（整套） | 1 |
| 一套衣服 | 1 |
| 银质高脚杯 | 1 |
| **总价值** | **约2,500–4,200 guilders** |

### 关于这笔交易的争论
- **Mackay（1841年）**：将此记载为一笔真实交易
- **Krelage（1942年）与 Garber（2000年）**：认为这是小册子中用于说明 guilder 购买力的**修辞手法**，并非真实交易
- 真相可能介于两者之间

## 4. Admirael van der Eijck — "Gouda"的替代候选

### 外观描述
- **类型**：Bizarden 类——黄色底色上分布深红/棕色火焰纹
- **文献充足**：见于 Pieter Cos 1637年目录
- 1637年2月5日以 **1,045 guilders** 成交
- 属于较昂贵但非最昂贵的品种之列
- 一颗"高档中档"球茎——作为游戏中"Gouda"的替代品颇为合适

## 5. "碎色"郁金香的生物学原理

### Tulip Breaking Virus（郁金香碎色病毒）

| 事实 | 详情 |
|---|---|
| **病原** | Tulip Breaking Virus（TBV），一种 potyvirus[^13]（马铃薯Y病毒属） |
| **传播途径** | 通过蚜虫传播 |
| **对花瓣的影响** | 产生多色、羽状、火焰状花纹 |
| **对植株的影响** | 进行性衰弱；子球产量减少 |
| **发现时间** | 直到20世纪20年代才被认识 |
| **17世纪的认知** | 被认为是一种令人向往的天然特性 |

### 这对价格意味着什么

1. **不可预测性**：每株碎色郁金香都独一无二——无法复制完全相同的花纹
2. **稀缺性**：病毒减少了繁殖，供给天然受限
3. **退化**：碎色球茎日益衰弱，最终死亡
4. **无法人为控制**：种植者无法诱发"碎色"——这一现象自然发生（通过蚜虫传播）
5. **地位象征**：每株碎色郁金香都是大自然创造的艺术孤品

### 现代状况

- 碎色郁金香如今**极为罕见**——为保护商业球茎作物，病毒已得到控制
- 一些传承碎色品种保存于植物园中
- 现代所谓"Rembrandt tulips"（伦勃朗郁金香）通过育种模拟碎色外观，但不含病毒

## 6. 郁金香贸易基础设施

### 种植季节历法

| 月份 | 活动 |
|---|---|
| **4–5月** | 花期。球茎不可挖掘或移动。仅能进行纸面交易。 |
| **6月** | 地上部分枯萎。球茎可以挖出并进行实物交易。 |
| **7–8月** | **现货市场**——实物球茎交易。即"Blank"市场。 |
| **9月** | 球茎须重新栽种。 |
| **10–11月** | 球茎入土。交易转向远期合约。 |
| **12–3月** | **投机高峰**——仅能进行远期合约（windhandel[^14]）。实物交割不可能。 |

### "郁金香信件"（Tulpenbrieven[^15]）

- 即郁金香球茎的远期合约
- 在公证人在见证下签署
- 载明品种、数量、价格和交割日期
- 许多合约针对的是尚在地里的球茎
- 在法律上不可强制执行（被视为赌博债务）

### 主要交易中心

| 城市 | 角色 |
|---|---|
| **Haarlem** | 郁金香种植与贸易的首要中心 |
| **Amsterdam** | 主要贸易中心，富商云集之地 |
| **Alkmaar** | 著名拍卖会举办地（孤儿院拍卖会，1637年2月5日） |
| **Utrecht** | 拥有活跃的 College[^16] |
| **Gouda** | 较小但确实存在的贸易中心 |
| **Leiden** | 有一定的贸易活动 |
| **Enkhuizen、Hoorn** | 次要贸易地 |

### 交易的实际运作方式

1. **现货交易**（夏季）：球茎实物交换
2. **远期交易**（冬季）：以纸质合约约定未来交割
3. **拍卖会**：有组织的交易活动，有时以慈善名义举办（如孤儿院拍卖）
4. **College 交易**：定期在酒馆举行的交易集会
5. **私人交易**：相熟的双方之间的私下买卖

## 7. 人们真的卖掉了自己的房子吗？

### 历史证据

**是的，但……**

- Pieter Cos 目录及当时的小册子记载了单颗球茎价值超过房屋的案例
- 一颗 Semper Augustus 价值5,000–6,000 guilders，确实超过一栋豪华运河宅邸（4,000–10,000 guilders）
- 一颗 Viceroy 价值4,200 guilders，相当于一栋运河宅邸的价格
- **然而**：Goldgar[^17] 发现，关于人们真的为购买郁金香而卖掉房产的证据十分有限
- 游戏角色 **Hendrik（卖掉房子梭哈郁金香）** 作为个别案例是说得通的，但不应被呈现为普遍现象

### 著名的"一栋房子换一颗球茎"之说

- 往往源自带有讽刺、夸张性质的小册子文学
- 在这些价格水平上的真实交易确实存在，但极为罕见
- 大多数交易为纸质合约，而非实物资产交换

## 8. 啤酒真的是2 Stuivers 一杯吗？

### 评估

- **是的，大致如此**——每杯2 stuivers[^18]（0.1 guilders）是一个合理的估算
- 这一价格也用于游戏的 `historical-data.json` 数据文件中
- 啤酒在当时廉价且人人饮用
- 一桶啤酒的价格约为8–12 guilders
- 游戏中 Maria 角色将啤酒定价为"两 Stuivers 一杯"是符合历史事实的

## 注释

[^1]: **Pieter Cos**：17世纪荷兰花商，于1637年编纂了著名的郁金香花木目录"Verzameling van een meenigte tulipaanen"（《众多郁金香汇编》），记录了当时的品种名称与交易价格，是研究郁金香狂潮的重要一手文献。
[^2]: **Couleren**：荷兰语，17世纪郁金香分类术语之一，指深色（紫色、栗色、红褐色）底色的郁金香品种群。此外还有 Rosen（白底红色火焰纹）、Violetten（白底紫色火焰纹）、Bizarden（黄底深色火焰纹）等分类。
[^3]: **Dumas**：即 Alexandre Dumas（大仲马），法国作家，于1850年出版小说《La Tulipe Noire》（《黑郁金香》），讲述了培育黑色郁金香的故事，使"寻找黑郁金香"成为著名的文化传说。
[^4]: **Guilder**（荷兰语 gulden）：荷兰盾，荷兰历史上的货币单位。在1630年代的郁金香狂潮时期，一个熟练工匠的年收入约200–400 guilders，一栋阿姆斯特丹运河宅邸的价格约4,000–10,000 guilders。1 guilder = 20 stuivers。
[^5]: **Rosen**：17世纪郁金香分类体系中的一个类别，指白色或浅色底色上带有红色或粉色火焰纹、羽毛纹的品种。Semper Augustus 即属于此类。该词源自"rose"（玫瑰），形容花瓣上如玫瑰般的红色纹路。
[^6]: **Tulip Breaking Virus**（郁金香碎色病毒，简称 TBV）：一种由蚜虫传播的植物病毒（属于马铃薯Y病毒科 Potyviridae），感染郁金香后会使花瓣出现火焰状、羽毛状的多色条纹。17世纪的人们不知道这是病毒引起的，将其视为令人向往的天然特征。该病毒同时会削弱植株，减少子球产量，最终导致植株死亡。直到20世纪20年代，其病毒本质才被科学界确认。
[^7]: **Adriaen Pauw**（1585–1653）：荷兰黄金时代最重要的政治家之一，曾任 Holland 大议长（Grand Pensionary），是荷兰共和国实际上的首相级人物。他拥有 Haarlem 附近的 Heemstede 庄园，以收藏珍贵的 Semper Augustus 郁金香而闻名。
[^8]: **Grand Pensionary of Holland**（Holland 大议长）：荷兰共和国时期 Holland 省的最高行政长官，实际上是联省共和国最有权势的政治职位之一。相当于现代意义上的政府首脑。
[^9]: **Violetten**：17世纪郁金香分类体系中的一个类别，指白色或浅色底色上带有紫色或紫罗兰色火焰纹的品种。Viceroy 通常被归入此类或 Bizarden 类。
[^10]: **Bizarden**：17世纪郁金香分类体系中的一个类别，指黄色底色上带有深色（红色、棕色、紫色）火焰纹的品种。Viceroy 有时被归入此类。
[^11]: **Last**：荷兰历史上的重量/容量单位，主要用于谷物贸易。1 Last 约等于2吨（约2,000公斤）。在 Pieter Cos 目录记载的著名易物交易中，Viceroy 球茎换得的物资即以 Last 计量。
[^12]: **Hogshead**：欧洲传统的液体容量单位，主要用于葡萄酒等酒类。1 Hogshead 约等于238–245升，因地区和历史时期有所差异。
[^13]: **Potyvirus**（马铃薯Y病毒属）：植物病毒中最大的一个属，通过蚜虫以非持久性方式传播。Tulip Breaking Virus 即属于此属。该属病毒可感染多种观赏植物和经济作物。
[^14]: **Windhandel**：荷兰语，字面意为"风中交易"，是当时对郁金香远期合约投机交易的讽刺性称呼。因球茎在冬季埋于地下无法交割，买卖双方仅凭一纸合约交易，如同在空中买卖风一般虚无缥缈。这一术语后来成为金融投机泡沫的经典隐喻。
[^15]: **Tulpenbrieven**：荷兰语，意为"郁金香信件"，即郁金香球茎远期交易合约。合约上载明品种、数量、价格和交割日期，由买卖双方在公证人在见证下签署。许多合约针对的是尚在土中无法挖出的球茎，在法律上被视为赌博债务，不可强制执行。
[^16]: **College**：17世纪荷兰郁金香交易者的组织形式，类似于交易俱乐部。成员定期在酒馆集会进行郁金香交易，制定交易规则和礼仪。Haarlem、Amsterdam、Utrecht 等城市都有活跃的 College。这种酒馆交易是郁金香狂潮时期最典型的交易场景之一。
[^17]: **Anne Goldgar**：英国历史学家，伦敦国王学院早期现代史教授。其2007年出版的 *Tulipmania: Money, Honor, and Knowledge in the Dutch Golden Age* 一书对郁金香狂潮进行了深入的社会文化分析，挑战了许多关于狂潮的传统叙事，指出实际参与人数有限、价格极端的案例被夸大。
[^18]: **Stuiver**（斯图弗）：荷兰历史上的硬币单位，1 guilder = 20 stuivers。Stuiver 是日常小额交易中最常用的货币单位之一，购买一杯啤酒、一块面包等均以 stuivers 计价。2 stuivers 约等于0.1 guilders。
[^19]: **South Sea Bubble**（南海泡沫）：1720年发生于英国的金融投机事件。South Sea Company 获得了与南美洲贸易的垄断特许权，股价在数月内暴涨数倍后崩盘，大量投资者倾家荡产。该事件与1637年郁金香狂潮、1720年 Mississippi Bubble 并称为早期三大金融泡沫。Maria 在游戏中提及南海公司属于时代错误，因为该事件比游戏设定的1637年晚了83年。
