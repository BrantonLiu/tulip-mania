# 1630年代阿姆斯特丹日常生活：历史研究报告

> 本研究报告编撰于2026年5月，服务于 Tulip Mania 1637 游戏项目。

---

## 1. 酒馆文化

### 荷兰酒馆（Herberg / Taveerne）

酒馆是1630年代荷兰社会与经济生活的核心场所，承担着多重功能：

- **郁金香投机交易的"交易大厅"**（即所谓的"Colleges"[^1]）
- **各阶层社交聚会的场所**
- **信息交汇中心**，新闻与流言在此传播
- **旅客住宿之处**
- **行会与各类组织的议事地点**

### 酒馆的面貌

- 通常为底楼大厅，天花板低矮
- 木桌木凳，殷实客人偶尔可以使用铺有桌布的座位
- 设有吧台或柜台，由店主斟酒待客
- 墙壁上常挂有绘画（这在荷兰的公共场所十分常见）
- 冬季设有壁炉或火塘取暖
- 有时另设后室供私密聚会使用（正是 Colleges 开会之处）
- 门外悬挂招牌，绘制酒馆名称图案（例如"金葡萄"）

### 酒馆供应的饮品

| 饮品 | 价格 | 备注 |
|---|---|---|
| **啤酒（bier）** | 每杯约2 stuivers[^2] | 最常见的饮品；荷兰啤酒享誉全欧 |
| **葡萄酒（wijn）** | 较贵 | 自法国、莱茵兰、西班牙进口 |
| **Jenever[^3]（荷兰杜松子酒）** | 价格不等 | 荷兰传统烈酒，以杜松子调味 |
| **Brandewijn（白兰地）** | 昂贵 | 蒸馏葡萄酒，属于奢侈品 |

### 1630年代的啤酒

啤酒是所有阶层的日常饮品——连儿童都饮用"淡啤酒"（small beer，低度数）。荷兰以啤酒闻名，阿姆斯特丹城内 Brewery（啤酒厂）林立。

- **啤酒常规价格**：每杯约2 stuivers（合0.1 guilders）
- **一桶啤酒**：售价约8-12 guilders
- **啤酒种类**：Hemp beer（hennepbier，麻啤）、Koyt beer、Jopen beer（酒精度更高，用于出口）
- 17世纪初的阿姆斯特丹约有70-100家啤酒厂

### Jenever（荷兰杜松子酒）

- 早在1500年代即已存在（并非如某些说法所称由 Franciscus Sylvius 发明）
- 以麦芽酒为基底，加入杜松子调味
- 最初用于药用
- 盛于郁金香形酒杯，斟至满溢
- **"Kopstootje"[^4] 习俗**：饮第一口时不持杯，需俯身就桌直接啜饮
- 阿姆斯特丹是 Jenever 的主要产地之一

### 酒馆老板的角色

酒馆老板通常是受社区尊重的人物，其职责包括：

- 主持聚会，充当非正式的文书
- 见证契约签署（包括郁金香交易）
- 向常客提供赊账
- 作为新闻与消息的来源
- 在激烈的交易场合维持秩序

## 2. 货币与购买力

### Guilder 体系

| 单位 | 价值 |
|---|---|
| **1 Guilder（gulden/florijn）**[^5] | 20 stuivers |
| **1 Stuiver** | 16 penningen（或8 duiten） |
| **1 Penning** | 2 duiten |

### 流通中的常见硬币

- **Guilder（gulden）**：银币，基本计量单位
- **Stuiver**：银币，1 Guilder 的1/20
- **Dubbele stuiver**：合2 stuivers
- **Schelling**：合6 stuivers
- **Rijksdaalder**[^6]：合2.5 guilders（银币）
- **Lion dollar（leeuwendaalder）**[^7]：约2.4 guilders
- **Ducat**[^8]：约3.5-4 guilders（金币）

### 购买力参照表

| 物品/服务 | 价格（约数） | 备注 |
|---|---|---|
| **一杯啤酒** | 2 stuivers（0.1 guilders） | 标准价格 |
| **一条面包** | 约5 stuivers（0.25 guilders） | 因谷物种类而异 |
| **一磅黄油** | 约8-10 stuivers | |
| **一磅奶酪** | 约4-6 stuivers | 荷兰奶酪为日常食品 |
| **一条鲱鱼** | 约1-2 stuivers | 常见食物 |
| **技术工人年薪** | 约250-300 guilders | 木匠、石匠等 |
| **工匠年薪** | 约400-500 guilders | 熟练手艺人 |
| **阿姆斯特丹商人收入** | 约1,000-2,000 guilders/年 | 中上阶层 |
| **阿姆斯特丹普通住宅** | 约300-500 guilders | 手艺人住所 |
| **运河宅邸** | 约1,000-4,000 guilders | 因地段而异 |
| **豪华运河宅邸** | 约5,000-10,000 guilders | 最为气派的宅邸 |
| **VOC[^9] 股份** | 面值约100 guilders | 在交易所交易 |
| **Semper Augustus[^10] 球根** | 5,000-6,000 guilders | 峰值价格 |
| **一匹马** | 约50-100 guilders | |

### 现代等价换算

- **100 guilders（1630年代）≈ 6,000美元（2024年）**
- **1 guilder ≈ 60美元（2024年）**（基于购买力的粗略估算）
- **2 stuivers 的啤酒（0.1 guilders）≈ 6美元**

## 3. 住宅与建筑

### 阿姆斯特丹运河带（Grachtengordel）[^11]

著名的运河带正是在1630年代逐步修建的：

- **开工时间**：1613年（Herengracht、Keizersgracht、Prinsengracht 三条运河）
- **大规模扩建**：1660-1665年（即"第四次扩展"）
- 1637年时，运河带尚处于部分完工状态——但许多宏伟宅邸已然落成

### 运河宅邸（Grachtenpanden）

- **面窄而进深大**——因为房产税按临街宽度征收
- **典型面宽**：6-7米
- **典型进深**：20-30米
- **多层结构**：4-5层，外加阁楼与地下室
- **显著特征**：
  - 临街一面设有阶梯形或钟形山墙（gable）
  - 顶部装有起重梁（hijsbalk），用于吊运家具上楼
  - 窗户宽大（荷兰人以玻璃工艺闻名）
  - 常见"voorhuis"（前屋）与"achterhuis"（后屋）的格局，中间设有庭院

### 室内布局

- **底楼（bel-etage）**：正式会客厅、办公间
- **楼上各层**：卧室、家庭生活空间
- **地下室（kelder）**：厨房、储物间、仆人住处
- **阁楼（zolder）**：储物间，有时亦作仆人住处

### 取暖方式

- 部分房间使用**瓷砖炉**（kachel）
- 主要房间设有**壁炉**
- **泥炭（turf）**是主要燃料——荷兰拥有大面积泥炭沼泽
- 煤炭虽有使用，但并不常见

### 普通民众的住所

- **工人与手艺人**：小住宅，有时仅有2-3个房间
- **许多人租房居住**——阿姆斯特丹有庞大的租赁市场
- **济贫院（hofjes）**：为老人和贫民提供的慈善住房

## 4. 荷兰社会中的女性

### 法律地位

在那个时代，荷兰女性拥有**显著较强的法律权利**：

- **寡妇**可以继承财产，并对财务与遗产保持控制权
- **已婚女性**的权利在夫权制度下受到更多限制，但荷兰法律比大多数欧洲国家更为有利
- **女性可以拥有财产**——1530年至1811年间，阿姆斯特丹有超过13,500名女性拥有房产
- **女性可以经营商业**——许多店主和商人都是女性
- **女性与男性一同领圣餐**——这在欧洲教会中并不常见

### 经济参与

荷兰女性的**经济活跃度**高于欧洲其他国家的同侪：

- 许多女性经营店铺、客栈和酒馆
- 女性参与市场交易与商业活动
- 寡妇通常继续经营亡夫的生意
- 部分女性参与了郁金香交易

### 与游戏角色的关联

**Anna（寡妇，35岁）** 这一角色的历史合理性：

- 寡妇继承亡夫的郁金香持仓——符合史实
- 寡妇自行管理财务——符合史实
- 寡妇对投机持谨慎态度——符合史实
- 她对子女未来的忧虑——符合史实

**Maria（酒馆老板娘，45岁）** 这一角色同样合理：

- 女性经营酒馆在当时十分常见
- 酒馆老板娘自然对街坊传闻和交易信息了如指掌

## 5. 饮食

### 1630年代荷兰人的日常饮食

**早餐**：面包涂黄油或奶酪，有时喝粥，配啤酒
**午餐**：正餐——面包、鱼或肉、蔬菜、啤酒
**晚餐（傍晚）**：较轻简——面包、奶酪、汤、啤酒

### 常见食物

| 类别 | 内容 |
|---|---|
| **面包** | 黑麦面包（最常见）、小麦面包（较贵） |
| **乳制品** | 黄油、奶酪（Edam、Gouda、Leiden） |
| **鱼类** | 鲱鱼（国民主食）、鳕鱼、鳗鱼、鲟鱼 |
| **肉类** | 猪肉、牛肉、羊肉、禽肉（较贵） |
| **蔬菜** | 卷心菜、胡萝卜、芜菁、洋葱、豌豆、豆角 |
| **水果** | 苹果、梨、樱桃、浆果 |
| **香料** | 胡椒、肉桂、肉豆蔻、丁香（通过 VOC 贸易获得） |
| **饮品** | 啤酒（人人皆饮）、葡萄酒（富人饮用）、Jenever、酪乳 |

### 荷兰鲱鱼

- 鲱鱼渔业是荷兰经济的基石
- **Hollandse Nieuwe**（新鲱鱼）——当季首批捕获，被视为珍馐
- 鲱鱼经腌制和醋渍以便保存
- 即使穷人也能经常食用鲱鱼

### 香料贸易

- VOC 从亚洲运来异域香料
- 胡椒、肉桂、肉豆蔻、丁香逐渐走入寻常百姓家
- 香料仍然昂贵，但在中产阶层厨房中日益常见
- 阿姆斯特丹是欧洲的香料贸易之都

## 6. 服饰与时尚

### 整体风格

- **深色、实用的着装**是荷兰人的普遍取向——这反映了 Calvinist[^12] 的价值观
- **黑色**极为流行（而且优质黑色染料价格不菲）
- **白色褶饰领/领圈（lieven）**[^18]——荷兰人标志性的领饰
- **禁奢法令**：较其他地区宽松；荷兰市民质朴的穿着更多出于自觉选择

### 男装

- 深色短上衣（doublet/马甲）
- 及膝裤（breeches）
- 白色亚麻领饰/褶领
- 长袜（多为黑色或深色）
- 宽檐帽
- 寒冷时披斗篷

### 女装

- 深色紧身上衣/系带衫
- 长裙（多为深色）
- 白色围裙（劳动妇女佩戴）
- 亚麻软帽（muts）遮盖头发（已婚妇女佩戴）
- 富裕女性佩戴珠宝（珍珠尤为流行）

### 服饰中的社会区分

- **富裕阶层**：上等面料（丝绸、精纺羊毛）、蕾丝镶边、金银纽扣、珠宝
- **中产阶层**：品质尚可的羊毛和亚麻、整洁体面
- **劳动阶层**：粗布面料、简单剪裁、缝补再缝补

## 7. 交通

### 阿姆斯特丹城内出行

- **步行**：大多数人的主要出行方式
- **运河驳船（trekschuit）**[^13]：运河上的马拉驳船——公共交通工具
- **划船**：用于穿越运河和 Amstel 河
- **马拉马车**：富人使用
- **骑马**：用于较远距离
- **船舶**：城际出行（非常普遍——荷兰是水路之国）

### 城际交通

- **定期驳船航线**连接荷兰各大城市
- 阿姆斯特丹至 Haarlem：驳船约2-3小时
- 阿姆斯特丹至 Leiden：约4-5小时
- 阿姆斯特丹至 Utrecht：约8-10小时
- 这些驳船按时刻表运行，是当时主要的公共交通工具

## 8. VOC（荷兰东印度公司）与日常生活

### VOC 在阿姆斯特丹的存在

- 成立于**1602年**，首次公开募股筹集了644万 guilders
- **股份在阿姆斯特丹交易所交易**——世界上第一个股票市场
- **股息**：每股面值的约10-12.5%（每年派发）
- **每股面值**：约100 guilders
- 峰值总市值：7,800万 guilders

### 普通民众与 VOC 的关联

- 许多阿姆斯特丹家庭至少有一人与海上贸易有关
- 有资金者均可买卖 VOC 股份
- VOC 带来的商品出现在全城各处商铺
- VOC 船只的启航与抵港是全城的盛事
- VOC 是市民自豪感与财富的来源

### VOC 与游戏的关联

游戏中将"航海股份"（即 VOC 股份）作为可交易资产：

- **符合史实**：VOC 股份确实被活跃交易
- **每股约100 guilders 的定价**是合理的
- VOC 股份与郁金香投机形成对照——它们是"真正"的投资
- 即便在郁金香泡沫破裂之后，VOC 依然持续繁荣

## 9. 宗教与社会背景

### 荷兰改革宗教会（Dutch Reformed Church）

- 荷兰共和国的**主导**教派
- 信奉**加尔文宗**（Calvinist）神学
- 多数人参加此教会，但**并非强制**——宗教宽容是荷兰的显著特征
- **对投机的态度**：复杂——加尔文宗推崇勤勉与节俭，对赌博持怀疑态度
- **对郁金香狂潮的态度**：许多牧师谴责其为赌博与虚荣

### 宗教宽容

- 荷兰共和国以相对的宗教宽容著称
- **天主教徒、犹太教徒、路德宗信徒、Mennonites[^14]、Remonstrants[^15]** 均可进行宗教活动（有时需低调进行）
- 阿姆斯特丹有规模可观的**犹太社区**（来自葡萄牙/西班牙的 Sephardic 犹太人）
- **Mennonites** 在郁金香交易者中所占比例畸高

### 社会习俗

- **握手**即表示成交（在郁金香交易中尤为重要）
- **共饮**巩固商业关系
- **共餐**具有社交属性
- **周日教堂礼拜**是常态（即使并非严格执行）
- **集市日**是重要的社交活动

## 10. 游戏对话中需注意的历史时代错误

### 不应出现的内容（属于时代错误）

1. ~~"Stock market crash"（股市崩盘）之类术语——当时尚无此概念~~
2. ~~"Investment portfolio"（投资组合）——现代概念~~
3. ~~"Leverage"（杠杆）作为金融术语——概念存在，但并非当时用语~~
4. ~~提及"南海泡沫"（South Sea Bubble，发生于1720年，当时尚未发生）~~
5. ~~"房产经纪人"或"股票经纪人"——这些职业在当时并不存在~~

### 符合时代背景的内容

1. 远期合约（forward contracts）、Windhandel[^16]、Tulpenbrieven[^17]（郁金香信函）
2. Guilder/stuivers/penningen 货币体系
3. 啤酒、葡萄酒、Jenever 作为酒馆饮品
4. 运河宅邸、trekschuit 驳船、泥炭取暖
5. 行会制度、公证人、孤儿法庭（orphan chamber）
6. VOC 股份、股息、交易所（Beurs）
7. 加尔文宗对投机的道德忧虑
8. 以社会荣誉与信任作为约束机制

## 注释

[^1]: **Colleges**：指1630年代荷兰酒馆中自发组织的郁金香交易聚会。交易者在固定的酒馆中定期集会，进行球根的买卖与竞价，是郁金香狂潮中非正式交易网络的核心形式。
[^2]: **Stuiver**：荷兰历史上的小额硬币单位，1 stuiver 等于1/20 guilder。相当于日常交易中最常用的计价单位，类似于中国旧时的"角"。
[^3]: **Jenever**：荷兰传统杜松子酒，以麦芽酒为基底、加入杜松子等植物调味。是英国金酒（gin）的前身。"Jenever"这个名字本身即来自荷兰语中的"杜松子"（jeneverbes）。
[^4]: **Kopstootje**：字面意为"小头顶撞"，是荷兰一种传统的 Jenever 饮用方式。将郁金香形酒杯斟满至表面张力形成的凸起，饮者不持杯，俯身用嘴直接啜饮第一口，随后配一口啤酒。这一习俗延续至今。
[^5]: **Guilder（gulden/florijn）**：荷兰历史上的基础货币单位，也译作"盾"。在17世纪是欧洲最重要的货币之一。gulden 为荷兰语原名，florijn 是其别称，源自佛罗伦萨的 fiorino 金币。
[^6]: **Rijksdaalder**：荷兰大型银币，面值2.5 guilders。名称字面意为"帝国塔勒"，源自德语区的 thaler 银币传统。
[^7]: **Lion dollar（leeuwendaalder）**：狮子塔勒，一种印有狮子图案的荷兰银币，广泛用于国际贸易。美国早期 dollar 一词的词源之一即来自此类 coins。
[^8]: **Ducat**：达克特金币，原为威尼斯发行的金币标准，后广泛流通于欧洲。在荷兰也是高面值的流通金币。
[^9]: **VOC（Vereenigde Oost-Indische Compagnie）**：荷兰东印度公司，成立于1602年，是世界历史上第一家发行股票的跨国贸易公司。其阿姆斯特丹证券交易所也是世界上第一个股票交易所。
[^10]: **Semper Augustus**：17世纪荷兰最珍贵的郁金香品种之一，以其火焰状的红白条纹花瓣闻名。名称为拉丁语，意为"永远的奥古斯都"。在郁金香狂潮高峰期，单颗球根曾以数千 guilders 的天价成交。
[^11]: **Grachtengordel**：阿姆斯特丹运河带，指环绕市中心的三条主要运河（Herengracht、Keizersgracht、Prinsengracht）及其两岸的宅邸群，是荷兰黄金时代城市建设的杰作，现为世界文化遗产。
[^12]: **Calvinist（加尔文宗）**：新教改革者约翰·加尔文（John Calvin）创立的基督教教派。强调勤俭、节欲、努力工作，认为世俗职业也是一种"天职"（calling）。加尔文宗是荷兰共和国的主流宗教，深刻影响了荷兰人的价值观与生活方式。
[^13]: **Trekschuit**：荷兰特有的马拉运河驳船，是17世纪荷兰最重要的公共交通工具。由沿岸步道上的马匹牵引，按固定时刻表运行，连接各主要城市。票价低廉，各阶层人士均使用。
[^14]: **Mennonites（门诺派）**：新教再洗礼派的一支，以和平主义和简朴生活著称。在荷兰享有宗教宽容，许多门诺派信徒是成功的商人和手工业者，其中不少人活跃于郁金香交易市场。
[^15]: **Remonstrants（抗辩派）**：荷兰新教内部的一派，由 Jacobus Arminius 的追随者组成。主张温和的预定论观点，与正统加尔文宗发生冲突，1618-1619年的多特会议（Synod of Dort）后一度遭到压制。
[^16]: **Windhandel**：字面意为"风中交易"，是荷兰人对郁金香投机中纯纸上交易（无实物交割）的蔑称。指买卖双方仅凭一纸合同或承诺书便进行转手倒卖，球根本身从未实际易手，如同在风中买卖虚无。
[^17]: **Tulpenbrieven**（郁金香信函）：郁金香狂潮时期使用的球根交易凭证或合同书。由于球根在土中无法交付，交易者以书面凭证代替实物进行买卖，是现代金融衍生品的早期雏形之一。
[^18]: **Lieven**：荷兰语中指17世纪流行的白色褶饰领圈，是荷兰黄金时代男女服饰中最具辨识度的配饰之一。在伦勃朗、弗美尔等荷兰画家的肖像画中屡见不鲜。其材质从普通亚麻到精美蕾丝不等，是社会地位的重要标识。
