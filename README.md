# Tulip Mania 1637 - 郁金香狂潮

一个关于1637年荷兰郁金香泡沫的历史模拟游戏。

## 游戏概述

玩家扮演1637年荷兰阿姆斯特丹的一名花商，在5天内体验郁金香市场的疯狂涨跌，体验历史上第一次资产泡沫的刺激与残酷。

## 游戏特色

- **历史沉浸**：基于真实历史事件，还原17世纪荷兰酒馆氛围
- **Galgame风格对话**：5个NPC角色，各具个性，每日随机对话
- **经济模拟**：真实的交易机制，包括价格波动、手续费、滑点
- **视觉体验**：复古风格UI，羊皮纸质感，烛光效果
- **多结局**：根据最终资产获得不同结局

## 游戏玩法

1. **初始资金**：10,000荷兰盾
2. **交易规则**：
   - 买入郁金香球茎、房产契约、航海股份
   - 卖出持有的资产
   - 每次交易收取1%手续费
   - 大额交易可能触发滑点（±5%）
3. **天数推进**：
   - Day 1-4：价格持续上涨
   - Day 5：泡沫破裂，价格暴跌80-95%
4. **NPC互动**：
   - 玛丽亚（酒馆老板娘）：热情友善
   - 科内利斯（老油条花商）：冷静理性
   - 亨德里克（赌徒）：狂热贪婪
   - 安娜（谨慎寡妇）：保守精明
   - 神秘商人：神秘精明

## 结局系统

- **传奇操盘手**（资产 > 200%）：在泡沫中全身而退还大赚一笔
- **幸运的旁观者**（资产 100-200%）：保住了本金
- **伤痕累累**（资产 50-100%）：损失了一部分
- **破产的花商**（资产 < 50%）：把一切都输了

## 游戏引擎

### 交易引擎
- **TradingEngine**：处理买入/卖出逻辑，计算手续费和滑点
- **PriceEngine**：计算每日价格波动，模拟市场情绪
- **DayEngine**：推进天数，触发价格更新和游戏阶段变化

### 对话引擎
- **DialogueEngine**：管理NPC数据和基础对话逻辑
- **DialogueLoader**：加载对话数据，根据天数和情绪选择对话

### 状态管理
- **GameState**：全局游戏状态
- **PlayerState**：玩家资产和交易历史
- **PriceHistory**：价格历史记录
