# PRD — 郁金香狂潮 Tulip Mania 1637 产品需求文档 v0.1

## 1. 产品定位

**郁金香狂潮 Tulip Mania 1637** 是一个以1637年荷兰郁金香泡沫为背景的Galgame风格交易模拟器。玩家扮演阿姆斯特丹花商Cornelis de Wit，在酒馆里通过对话推进天数，体验泡沫形成到破裂的全过程。

核心体验：**Galgame对话式操作 → 快速天数推进 → 价格疯涨/暴跌的刺激 → 交易决策快感**

目标：黑客松demo，2分钟内体验完主流程，传达"泡沫不可持续"的主题。

## 2. 目标用户

| 用户画像 | 特征 |
|---|---|
| 核心用户 | 喜欢Galgame/视觉小说的玩家，对历史金融题材感兴趣 |
| 黑客松评委 | 想看到有创意、有完成度、有技术亮点的demo |
| 泛用户 | 通过社交媒体分享进入，想快速体验的普通用户 |

## 3. 用户流程

```
开场动画（1637年2月背景介绍）
  → 身份介绍（你是花商Cornelis de Wit）
    → 酒馆主界面（Day 1）
      → 与NPC对话触发交易提示
        → 买入/卖出决策
          → 推进到下一天（Day 2-4，重复交易）
            → Day 5 泡沫破裂
              → 结局页面（财富结算 + 不同结局）
                → "历史上的今天"真实历史介绍
```

**时长控制**：
- 开场：15-20秒
- Day 1-4交易：每个15-20秒（含对话+交易+天数推进），总约60-80秒
- 泡沫破裂+结局：20-30秒
- 总计：约95-130秒（2分钟内）

## 6. 交易系统设计

### 资产清单

| 资产 | 类型 | 基础价格 | 特点 |
|---|---|---|---|
| **红色郁金香 (Semper Augustus)** | 郁金香 | 1000金币 | 最贵，涨幅最大，跌幅最大 |
| **黄色郁金香 (Gouda)** | 郁金香 | 500金币 | 中档，涨跌中等 |
| **紫色郁金香 (Viceroy)** | 郁金香 | 300金币 | 低档，涨跌较小 |
| **黑色郁金香 (Black Tulip)** | 郁金香 | 800金币 | 稀有品种，涨幅大但有随机性 |
| **房产契约** | 房地产 | 2000金币 | 稳定增值5-10%/天，波动小 |
| **航海股份** | 股票 | 1500金币 | 稳定但无聊，3-8%/天，低风险 |

### 价格机制

**Day 1-4（泡沫形成期）**：
- 郁金香每天涨50-200%（随机）
- 房产每天涨5-10%（稳定）
- 航海股份每天涨3-8%（稳定）

**Day 5（泡沫破裂）**：
- 郁金香暴跌80-95%（恐怖）
- 房产跌20-30%
- 航海股份跌10-20%

**波动模拟**：
- 每个品种每天有 ±20% 的随机波动
- 黑色郁金香有30%概率涨幅翻倍，也有20%概率突然暴跌

### 交易规则

- 初始资金：10000金币
- 手续费：每次交易收取1%手续费
- 最小交易单位：1份合约
- 滑点：大额交易（>总资产30%）触发 ±5% 滑点
- 不支持做空（历史真实情况）

## 7. 天数推进机制

### 天数结构

- **Day 1**：泡沫刚开始，价格温和上涨（50-80%）
- **Day 2**：泡沫加速，价格大幅上涨（80-150%）
- **Day 3**：泡沫疯狂，价格暴涨（100-200%）
- **Day 4**：泡沫顶峰，价格最后疯狂（150-200%）
- **Day 5**：泡沫破裂，价格暴跌（80-95%）

### 天数推进操作

1. 玩家点击"推进到下一天"按钮
2. 游戏结算当日盈亏（计算未平仓资产的市值变化）
3. 生成新价格（根据天数和价格算法）
4. 触发NPC对话事件（随机选择一个NPC）
5. 更新UI显示

### 触发对话的场景

- 每天推进后随机触发1-2个NPC对话
- 价格大幅涨跌时触发特定NPC对话
- 泡沫破裂（Day 5）触发所有NPC恐慌对话

### 价格看板 (PriceBoard)
- 黑板背景，粉笔字风格
- 显示所有资产的当前价格
- 红色显示（涨价）、绿色显示（跌价）
- 箭头指示涨跌幅（↑ +50%）

### 交易面板 (TradePanel)
- 资产选择下拉框
- 买入/卖出按钮
- 数量选择（- / 1 / +）
- 确认按钮
- 预计交易金额提示

### 资产面板 (AssetPanel)
- 当前资金
- 持有资产列表
- 总资产价值

### 天数控制 (DayControl)
- "推进到下一天"按钮
- 天数显示（Day 1/5）
- 进度条

### 结局场景 (EndingScene)
- 显示玩家最终财富
- 根据财富区间给出不同结局（5个）
- 显示交易历史摘要
- "历史上的今天"历史资料卡片
- "重新开始"按钮

## 9. 结局系统

### 结局判定

| 最终财富 | 结局名称 | 描述 |
|---|---|---|
| > 50000 | **泡沫之王** | 你是泡沫的王者！在崩溃前完美套现，成为阿姆斯特丹最富有的花商。但历史不会记住你，因为你的财富建立在泡沫之上。 |
| 30000-50000 | **精明投机者** | 你及时止损，保住了大部分利润。你是聪明人，知道什么时候该离场。但你也会永远记得那段疯狂的时光。 |
| 15000-30000 | **普通商人** | 你赚了一些钱，也亏了一些。泡沫破裂时你还在场，但幸运的是没有倾家荡产。这大概就是普通人的命运。 |
| 5000-15000 | **受害者** | 你相信了"这次不一样"，结果血本无归。你只是泡沫中无数普通人的缩影。历史不会记得你，但你永远会记得这一天。 |
| < 5000 | **破产者** | 你把一切都押在了郁金香上，结果一夜之间一无所有。泡沫破裂了，但你的人生也破裂了。这可能是历史上最昂贵的教训。 |
